Проект: Elevator
Версия Python: 3.12.7
Библиотеки:(tkinter, threading, time, queue).


Скачайте папку с программой.


Запуск на Windows:
1. Убедитесь, что у вас установлен Python 3
2. Открыть терминал/командную строку в папке проекта.
3. Выполнить команду: python Elevator_code.py (или запустить файл через любую IDE).
4. Следуйте инструкциям во всплывшем окне 

Запуск на Linux Ubuntu:
1. Установите библиотеку командой:
	sudo apt-get update
	sudo apt-get install python3-tk
2. После установки запустите скрипт:
	python3 Elevator_code.py
